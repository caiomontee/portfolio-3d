# 🚀 3D Developer Portfolio

Um portfolio 3D moderno e animado construído com React, Three.js, TailwindCSS e efeitos de movimento — desenvolvido para destacar suas habilidades de forma criativa.

## ✨ Funcionalidades

- 🔥 Visuais 3D com **React Three Fiber** e **Drei**
- ⚡ Transições suaves e animações de scroll com **Framer Motion**
- 🎨 UI responsiva e limpa com **TailwindCSS**
- 💌 Formulário de contato funcional
- 🧱 Componentes de interface com **Aceternity UI** e **Magic UI**
- 🚀 Desenvolvimento rápido com **Vite**

---

## 🛠 Tecnologias

| Tecnologia        | Descrição                                             |
|-------------------|-------------------------------------------------------|
| React             | Biblioteca JavaScript para o Front-end                |
| Vite              | Ambiente de desenvolvimento e bundler rápido          |
| TailwindCSS       | Framework CSS utilitário                              |
| React Three Fiber | Renderização 3D com Three.js no React                 |
| Drei              | Helpers e abstrações para R3F                         |
| Framer Motion     | Biblioteca de animações para React                    |
| Aceternity UI     | Componentes de UI customizados                        |
